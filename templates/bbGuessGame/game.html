BattleBordle is a BattleBots guessing game inspired by a concept posted to OOTA by @bravadobattlebot. Enter a robot into the search bar and click to guess, for each category you will get either a green square for a match, a yellow square for a near miss or a grey square for a complete miss.
Info in categories are correct according to the latest appearance on the show. For example Chomp is a green and yellow axe, not a silver crusher and Red Devil is considered Canadian.
For the sake of keeping things fun (and not making the game impossible) BattleBordle ONLY contains data from the main series of reboot battlebots, no side events / live shows included. The categories are as follows:

First Letter: Does the first letter of your guess match the first letter of today's hidden BattleBot.

Debut: Which season of the reboot did the robot compete at, a yellow is awarded for the seasons either side of the correct answer.

Weapon*: What is the primary weapon type of the robot. A weapon in the same rough category (eg drum spinner / vertical spinner) will be yellow. (As an edge case, Fusion is classed as a vert here)

Best Finish: Where in the overall tournament did this robot finish? Categories are: champion, runner-up, semi-finalist, quarter-finalist, round of 16, round of 32, qualifiers/pre-qualifers/fight nights, alternate/reserve. Yellows are again awarded for 1 either side of the correct answer.

Color: What color a robot is can be quite subjective, to get around this as well as possible each robot here has been assigned a primary and (mostly) a secondary color. Matching one gives a yellow square and both a green, although this might still be out sometimes depending on if you think that's a blue or a green etc. For certain very multi-colored bots where a single color is hard to determine they've been assigned the elusive "rainbow" color for a color slot.

Country: Which country is this robot from? In the event in which robots are listed as being from multiple countries the first country listed on the wiki is chosen. (And as an edge case, Cobalt is considered British)
